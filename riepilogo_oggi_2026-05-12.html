2. Sezione Attività · 2 nuovi Quadri di Stampa

Due nuovi quadri stampabili per la rendicontazione del club nell'anno sociale Lions (1 luglio → 30 giugno).

2.1 Helper anno sociale (src/lib/anno-sociale.ts)

Modulo dedicato per il calcolo dell'anno fiscale Lions, con auto-detect dell'anno corrente in base alla data odierna:

// Esempio: oggi 12/05/2026 → siamo in FY 2025/26
getCurrentAnnoSocialeStart()   // → 2025
getAnnoSocialeRange(2025)       // → { from: '2025-07-01', to: '2026-06-30', label: '2025/26' }
getRecentAnniSociali(8)         // → ultimi 8 anni sociali per il select

2.2 Quadro A: tutte le attività del club nell'anno sociale

Route: /attivita/quadri/club-anno

L'utente seleziona un club e un anno sociale (default: corrente). Vengono mostrate tutte le attività del club in quel periodo, ordinate per data crescente.

2.3 Quadro B: Amministrazione vs Service nell'anno sociale

Route: /attivita/quadri/club-anno-amm-service

Stesse colonne del Quadro A, ma con una differenza chiave: Persone Servite - Limite Massimo (persone_servite_limite) al posto di Persone servite.

Le attività vengono divise in due tabelle separate:

  • SERVICE — tutte le attività con causa ≠ 'Amministrazione' (eventi, raccolte fondi, service umanitari, ecc.)
  • AMMINISTRAZIONE — le attività con causa = 'Amministrazione' (Direttivi, Riunioni di Club, ecc.)

Ogni tabella ha il proprio subtotale, e in fondo un riquadro evidenziato con i Totali Complessivi (Service + Amministrazione). Stampa landscape.

2.4 Stili print landscape

Aggiunta classe .print-landscape in globals.css con regola @page { size: A4 landscape; } via page: landscape-page. Applicata ai due nuovi quadri attività.

3.5 Header e logica FY/PFY

Header dashboard: "Anno sociale 2025/26 (FY) vs 2024/25 (PFY)". Il calcolo è automatico:

// 1 luglio → 30 giugno
fyStart = month >= 6 ? year : year - 1   // 12 mag 2026 → fyStart = 2025
fy = { from: '2025-07-01', to: '2026-06-30', label: '2025/26' }
pfy = { from: '2024-07-01', to: '2025-06-30', label: '2024/25' }
